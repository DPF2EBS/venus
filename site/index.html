Venus Introdution

Venus is a flexible, modular JavaScript library designed for all the JavaScript developer.It give you a powerful solution for data visualization.Venus is builded on KineticJS and Raphaël.

KineticJS is an HTML5 Canvas JavaScript library that extends the 2d context by enabling high performance path detection and pixel detection for desktop and mobile applications.

Raphaël is a small JavaScript library that should simplify your work with vector graphics on the web. If you want to create your own specific chart or image crop and rotate widget, for example, you can achieve it simply and easily with this library.

Venus currently supports Firefox 3.0+, Safari 3.0+, Chrome 5.0+, Opera 9.5+ and Internet Explorer 6.0+.

Venus Developers: quanhua.ma, zhi.cun, xinwei.wang, yongliang.li, chao.wang @dpf2e-datavisualization-team.
